11. Structures a web page

Document and website structure
  1. header - present introduction of element
  2. main - main contain of the page, use only one
  3. nav - link or menu
  4. aside - content related to main content
  5. footer - same as header but places last part of web page
  6. article - encloses a block of related content that makes sense on its own without the rest of the page (e.g., a single blog post).
  7. section - group related element
